Github

GitHub është një platformë online ku programuesit ruajnë, ndajnë dhe bashkëpunojnë në kod.

Per qka perdoret Githubi

GitHub përdoret për disa gjëra shumë të rëndësishme në programim: 🔹 1. Ruajtja e kodit Mund të ruash projektet e tua (p.sh. lojëra në Python, faqe web, etj.) në një vend online që nuk humbet. 🔹 2. Bashkëpunimi me të tjerët Shumë persona mund të punojnë në të njëjtin projekt në të njëjtën kohë pa u ngatërru.
